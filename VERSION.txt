Widget ID: com.plasmawidget.cpumonitor
Target Platform: KDE Plasma 6.0+

Dependencies:

Changes in 1.3:
- Fix CPU frequency display (was showing 0 due to incorrect unit conversion)
- Sensor returns MHz, now correctly dividing by 1000 for GHz display
- Add enabled binding to sensors (reduces overhead when widget hidden)
- Improved error handling for undefined sensor values

Changes in 1.2:
- Uses ksysguard sensors for CPU/memory stats (lower CPU overhead)
- Reads /proc directly instead of psutil (no external dependency)
- Separate lightweight script for top processes
