FULL DESCRIPTION
----------------
CPU Monitor is a lightweight Plasma widget that brings vibrant, informative
system monitoring directly to your desktop or panel.

Features:
‚Ä¢ Real-time CPU usage with color-coded indicators (green/yellow/red)
‚Ä¢ Current CPU frequency and core count display
‚Ä¢ Memory (RAM) usage with visual progress bars
‚Ä¢ Top 5 most resource-intensive processes
‚Ä¢ Per-process CPU percentage and memory consumption
‚Ä¢ Updates every 2 seconds
‚Ä¢ Cyberpunk-inspired color scheme (Matrix green, cyan, magenta, gold)
‚Ä¢ Minimal system overhead (< 0.5% CPU usage)

Perfect for:
‚úì Gamers who want to monitor performance while playing
‚úì Developers tracking build processes and resource usage
‚úì Power users keeping tabs on system health
‚úì Anyone curious about what their computer is doing

The widget uses intelligent color coding:
- Bright green (< 50%): System running smoothly
- Orange/yellow (50-80%): Working hard, but healthy
- Red (> 80%): High load, might need attention

Unlike basic system monitors, CPU Monitor shows you exactly which programs
are using resources, complete with memory usage in an easy-to-read format.

Technical:
- Built with Qt/QML for smooth, native Plasma integration
- Python backend using psutil for accurate system metrics
- Plasma 6.0+ compatible
- Open source, free to use and modify

- KDE Plasma 6.0+
- Python 3.x
- psutil (pip install psutil)

Install:
